# Changelog

## 1.0.0

- Initial release
- Track profession cooldowns: Shadowcloth, Spellcloth, Primal Mooncloth, Transmute, Brilliant Glass, Salt Shaker
- Minimap button with draggable position
- Tooltip showing all characters' cooldowns with time remaining
- Login alerts for ready cooldowns on current character
- Automatic detection via spell cast, tradeskill window scan, and spellbook scan
- Slash commands `/chronocraft` and `/cc`
- Bilingual support: English and French (auto-detected)
- Account-wide SavedVariables for cross-character tracking
- Auto-purge of expired cooldowns (24h after expiry)
